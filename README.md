# 🌱 FarmIQ — AI-Powered Farming Assistant for Africa

FarmIQ is a mobile application that helps smallholder farmers in Ghana and across Africa make smarter farming decisions using Artificial Intelligence.

## 🚀 Features
- 🤖 AI Farming Chatbot (powered by Groq LLaMA 3.3)
- 🌤️ Weather Forecasting
- 🐛 Pest & Disease Detection
- 🌱 Crop Advisory
- 📊 Live Market Prices
- 💧 Smart Irrigation Guidance

## 🛠️ Tech Stack
- **Frontend:** React Native + Expo + TypeScript
- **Backend:** Spring Boot (Java) — Microservices
- **Database:** PostgreSQL
- **AI:** Groq API (LLaMA 3.3 70B)
- **Auth:** JWT Authentication

## 🌍 Target Market
Smallholder farmers in Ghana and across Africa.
